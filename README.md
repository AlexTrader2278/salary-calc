# Калькулятор зарплаты

Офлайн-калькулятор зарплаты для северной вахты: оклад по часам, ночные, праздничные, проф. мастерство, вредность, ежемесячная премия, районный коэффициент и северная надбавка, вахтовая надбавка, компенсация пути, отпускные по среднему, НДФЛ.

Один HTML-файл, всё считается на устройстве, ничего не отправляется в сеть. Встроен производственный календарь 2026 (нормы часов редактируются).

## Как пользоваться
- Открыть ссылку GitHub Pages в браузере телефона → меню → «Добавить на главный экран». Появится иконка, работает офлайн.
- Или просто скачать `index.html` и открыть локально.

## Состав
- `index.html` — приложение и все расчёты
- `manifest.webmanifest`, `sw.js`, `icon-*.png` — обвязка PWA (иконка + офлайн-кэш)
